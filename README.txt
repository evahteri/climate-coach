# Climate Coach

This is the final project made for the course User Interfaces and Usability

## How to run

1. Clone this repository.

2. Make sure you have flutter installed

3. In the root of the program folder, run ``` flutter run lib/main.dart ```

4. If the command line tool finds multiple devices, choose Chrome with number keys.

5. Close the program by closing the chrome window or by pressing Ctrl + C
